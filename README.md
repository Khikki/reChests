# Re:Chests

Re:Chests is a Minecraft Forge 1.7.10 mod that adds wood-specific chest variants, trapped chests, and chest minecarts.

## Features

- Oak, spruce, birch, jungle, acacia, and dark oak chests
- Matching trapped chest variants
- Matching chest minecarts for regular chest variants
- Custom single and double chest textures

## Development

- Minecraft: `1.7.10`
- Forge: `10.13.4.1614`
- Java: `8`

## Build

```powershell
.\gradlew.bat build
```

The built jar is written to `build/libs/`.
